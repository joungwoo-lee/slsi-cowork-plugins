# 개인 RAG 컨셉

완벽한 **"설치 ➔ 파일 투척 ➔ 클로드 스킬 호출"** 파이프라인을 완성해 드립니다.
말씀하신 흐름이 끊기지 않도록, 초기 셋업 시 **'Workspace(지식 베이스)'까지 API로 자동 생성**하고, 파일을 폴더에 넣은 뒤 **임베딩(벡터화)을 자동화하는 스크립트**까지 추가했습니다.

## 1단계: 셋업 실행

AnythingLLM 다운로드, API 고정키 삽입, 서버 구동, 전용 워크스페이스 생성까지 자동화합니다.

## 2단계: 문서 투척 및 임베딩

`~/my_rag_docs` 폴더에 파일을 넣고 업로드 + 임베딩 스크립트를 실행합니다.

## 3단계: 클로드 연동 및 스킬 호출

Claude Desktop MCP 설정으로 AnythingLLM 서버를 연결하고, `my_rag` 워크스페이스를 대상으로 검색합니다.
