첨부파일의 내용까지 마크다운으로 변환하여 검색 대상에 포함하는 조건을 반영하여, **최종 요구사항 명세서**를 다시 정리해 드립니다.
이제 이 시스템은 메일 본문뿐만 아니라 **첨부된 문서(PDF, Word 등)의 내부 텍스트까지 추출하여 키워드(SQLite)와 의미(Qdrant) 검색**이 가능하도록 설계됩니다.
### 📋 PST 기반 AI 하이브리드 검색 시스템 요구사항 (첨부파일 검색 포함)
#### 1. 운영 환경 및 제약 조건
 * **OS:** Windows 10/11 네이티브 (Docker/WSL 사용 안 함)
 * **언어:** **Python 3.9 (.exe 설치)**
   * *이점:* pypff 등 주요 라이브러리의 빌드 완료된 파일(Wheel)을 사용하여 **Microsoft C++ Build Tools 설치가 필요 없습니다.**
 * **제약 사항:** 아웃룩(Outlook) 실행 없이 백그라운드에서 PST 파일 직접 디코딩.
 * **모델 사용:** **외부 API 엔드포인트**를 호출하여 임베딩값(Dense Vector) 획득 (로컬 딥러닝 연산 최소화).
#### 2. 데이터 추출 및 마크다운 통합 변환 (ETL)
 * **PST 추출:** pypff 및 libratom을 사용하여 본문, 메타데이터, 첨부파일을 추출합니다.
 * **메일 본문 변환:** markdownify를 사용하여 HTML 본문을 마크다운으로 변환합니다.
 * **첨부파일 변환 (핵심):**
   * **PDF:** PyMuPDF(fitz)를 사용하여 텍스트를 추출하고 마크다운 형식으로 재구성합니다.
   * **Word(.docx):** python-docx를 사용하여 문서 내용을 마크다운으로 변환합니다.
   * **통합:** 변환된 첨부파일 텍스트를 메일 본문 마크다운 뒤에 **[첨부파일: 파일명.pdf]** 형태의 구분자와 함께 병합하여 하나의 통합 검색 문서로 만듭니다.
#### 3. 하이브리드 검색 엔진 (2-Track 시스템)
본문과 첨부파일 내용이 합쳐진 통합 마크다운 문서를 두 가지 DB에 인덱싱합니다.
**① 의미 검색 (Qdrant 로컬 모드)**
 * **담당:** 질문의 '의도'와 본문/첨부파일의 '내용'이 유사한 메일 찾기.
 * **방법:** 외부 API에서 받은 통합 문서의 **Dense 벡터**를 로컬 Qdrant 폴더(C:\VectorDB)에 저장.
**② 키워드 검색 (SQLite FTS5)**
 * **담당:** 특정 단어(예: 품번, 계약번호, 첨부파일 내 특정 키워드)가 정확히 포함된 메일 찾기.
 * **방법:** SQLite의 **Virtual Table(FTS5)** 기능을 사용하여 본문+첨부파일 통합 텍스트를 인덱싱.
#### 4. 데이터 저장 및 관리 구조
 1. **물리 저장소 (C:\Outlook_Data\Files\):**
   * [Mail_ID]/body.md (통합 마크다운 파일)
   * [Mail_ID]/attachments/ (원본 첨부파일 보관)
 2. **SQLite DB (C:\Outlook_Data\metadata.db):**
   * **Metadata 테이블:** 제목, 날짜, 발신자, 파일 경로 저장.
   * **FTS5 테이블:** 통합 마크다운 텍스트를 인덱싱하여 **본문+첨부파일 키워드 검색** 지원.
 3. **Qdrant 로컬 폴더 (C:\Outlook_Data\VectorDB\):**
   * 통합 문서의 **의미(Dense) 벡터** 저장.
### 🛠️ 추가 설치가 필요한 라이브러리 (CMD)
첨부파일 파싱을 위해 아래 라이브러리가 추가됩니다. 여전히 Python 3.9에서는 C++ 빌드 없이 설치 가능합니다.
```bash
# 기본 도구
pip install pypff-python libratom markdownify qdrant-client requests

# 첨부파일 파싱 도구 (PDF 및 Word)
pip install pymupdf python-docx

```
### 💡 최종 검색 시나리오
 1. **사용자 질문:** "작년 협력사 보안 점검 결과 보고서 내용 찾아줘."
 2. **검색 수행:**
   * **SQLite:** "보안 점검", "결과 보고서"라는 단어가 포함된 메일과 **첨부파일 내용**을 검색합니다.
   * **Qdrant:** 보안 점검 결과와 맥락이 유사한 내용을 본문 및 **첨부파일 텍스트**에서 찾아냅니다.
 3. **결과 도출:** Claude가 검색된 마크다운(본문+첨부파일 통합본)을 읽고, "어떤 메일에 첨부된 PDF 파일 몇 페이지에 이런 내용이 있습니다"라고 답변합니다.
